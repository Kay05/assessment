Ranking Changes Explained

No ranking changes occurred because the higher-ranked player won or the players were adjacent in a draw.

In a draw between non-adjacent players, the lower-ranked player moves up one position.

When a lower-ranked player beats a higher-ranked player:

  • The higher-ranked player moves down one position
  • The lower-ranked player moves up by half the original rank difference
  • Other players' ranks are adjusted to maintain sequential ranking
